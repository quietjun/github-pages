pages.json 자동 생성 스크립트 설정 안내 및 임베드 코드

① pages.json 생성 스크립트 — 설정 항목
설정 변수 목록
변수명 기본값 타입 설명
ROOT_DIR "." str 스캔 루트 디렉토리 — 저장소 루트 기준
OUTPUT_FILE "pages.json" str 생성할 출력 파일명
EXCLUDE_FILES {"index.html", "404.html"} set 트리에서 제외할 파일명 (정확히 일치)
EXCLUDE_DIRS {".git", ".github", "node_modules", "_site"} set 스캔에서 제외할 폴더명
EXCLUDE_PREFIX "_" str 이 문자로 시작하는 파일·폴더 전체 제외
SCAN_EXTENSIONS {".html", ".htm"} set 스캔 대상 확장자 — 해당 확장자만 트리에 포함
설정 예시 코드
Python — generate_pages.py 
# ── 설정 ──────────────────────────────────────────
ROOT_DIR        = "."                               # 스캔 루트 (저장소 루트)
OUTPUT_FILE     = "pages.json"                     # 출력 파일
EXCLUDE_FILES   = {"index.html", "404.html"}        # 제외할 파일명
EXCLUDE_DIRS    = {".git", ".github", "node_modules", "_site"}  # 제외할 폴더
EXCLUDE_PREFIX  = "_"                               # 이 문자로 시작하는 파일/폴더 제외
SCAN_EXTENSIONS = {".html", ".htm"}               # 스캔할 확장자
# ──────────────────────────────────────────────────
참고 사항
  • EXCLUDE_PREFIX 설정 시 파일명·폴더명 모두 적용 — 예: _draft/, _temp.html 모두 제외
  • 스크립트 실행 후 pages.json 이 저장소 루트에 생성 → Docs Viewer가 자동 로드
  • 폴더 구조가 그대로 트리로 반영 — 하위 폴더는 접히는 노드로 표시
  • 확장자 없이 디렉토리만 있는 경우 트리에 노출되지 않음
② 티스토리 임베드 코드 — 복사해서 사용
iframe 임베드 코드
  • 아래 코드를 티스토리 HTML 편집기에 붙여넣기
  • src 의 URL을 실제 문서 경로로 교체
  • myIframe id는 페이지 내 고유하게 유지
HTML — 티스토리 붙여넣기용 
<p><iframe src="https://quietjun.github.io/github-pages/ai/rnn/wordpiecetokenizer.html" id="myIframe" width="100%" frameborder="0" scrolling="no"></iframe></p>
