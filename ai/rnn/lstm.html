② LSTM 셀 구조 — 두 가지 메모리
Cell State (장기 기억)
Forget Gate (망각)
Input Gate (입력)
Candidate (후보값)
Output Gate (출력)
🟢
Candidate(후보값)는 tanh 함수를 통해 현재 입력에서 추출한 "기억 후보"예요.
값의 범위는 −1 ~ 1이며, 그 자체로 바로 기억에 더해지지는 않아요.
Input Gate(0 ~ 1)와 곱해져서 "이 후보 중 얼마나 실제로 기억할까?"를 결정한 뒤 Cell State에 반영돼요.
③ 게이트 동작 — 단계별로 살펴봐요
예시 문장 "나는 어제 학교에서 밥을 먹었다"의 마지막 토큰 "먹었다"를
LSTM 셀이 처리하는 흐름을 따라가봐요 🐣
