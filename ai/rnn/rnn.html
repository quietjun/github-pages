① RNN 셀 — 언롤링 개념
  • RNN은 하나의 셀을 토큰 수만큼 반복 실행하는 구조
  • 각 타임스텝마다 이전 은닉 상태 h를 입력으로 받아 순차 처리
  • 이를 시간 축으로 펼쳐서 표현한 방식 → 언롤링(unrolling)
RNN 셀 언롤링 (Unrolling)
② nn.RNN 파라미터
함수 시그니처
nn.RNN(input_size, hidden_size, num_layers=1, bidirectional=False, device=None)
input_size
  • 입력 벡터 1개의 차원 수
  • 임베딩 차원이 128 → input_size=128
hidden_size
  • 은닉 상태 벡터의 차원 수
  • 클수록 표현력 ↑, 파라미터 수 ↑
num_layers 기본값 = 1
  • RNN 셀의 수직 적층 수
  • 깊을수록 복잡한 패턴 학습 가능
bidirectional 기본값 = False
  • True → 순방향 + 역방향 동시 처리
  • 출력 크기: hidden_size × 2
