CLIP이란 무엇인가?

OpenAI · 2021 — 이미지와 언어를 함께 이해하는 AI

한 줄 요약 : 사람처럼 이미지와 텍스트를 동시에 이해해, "이 사진은 고양이야" 같은 연결을 스스로 학습한 모델
기존 문제 → CLIP의 해결
❌ 기존 방식의 한계

이미지 분류 AI는 고정된 카테고리만 인식
→ "강아지 / 고양이 / 자동차" 외 새 카테고리가 생기면 수천 장의 새 학습 데이터가 필요했음

→
✅ CLIP의 접근

이미지와 텍스트 설명을 같은 공간에 표현
→ 새 카테고리도 텍스트로 설명만 하면 추가 학습 없이 즉시 인식 가능

핵심 아이디어 — 같은 공간에 매핑
작동 원리 — 3단계
1
대규모 데이터 수집
인터넷에서 이미지 + 텍스트 설명 쌍 4억 개 수집
→ "고양이 사진" + 🐱 이미지 같은 형태
2
대조 학습
맞는 쌍 → 벡터 거리 가깝게 ↓
틀린 쌍 → 벡터 거리 멀게 ↑
→ 수십억 번 반복 학습
3
Zero-shot 활용
새 이미지 → 후보 텍스트들과 유사도 비교 → 가장 가까운 것 선택
→ 재학습 없이 바로 사용
Zero-shot 분류 예시
🐶
→
자동차 사진 하늘 사진 강아지 사진 ✓ 음식 사진 건물 사진
🌅
→
실내 사진 일몰 풍경 사진 ✓ 도시 야경 사진 인물 사진

→ 각 텍스트를 벡터로 변환 후 이미지 벡터와 유사도 계산 · 가장 높은 것 선택
→ 후보 목록만 바꾸면 어떤 분류도 즉시 가능 (재학습 불필요)

왜 CLIP이 중요한가?
🔓
범용성
특정 태스크에 종속되지 않음
→ 분류 · 검색 · 생성 등 다양하게 활용
💬
언어로 제어
새 개념을 텍스트로 설명만 하면 됨
→ 데이터 수집 · 라벨링 비용 대폭 감소
🧱
기반 모델
DALL·E 2, Stable Diffusion 등
이미지 생성 AI의 핵심 구성 요소로 활용
📏
평가 지표
CLIP Score → 생성된 이미지가
텍스트 설명과 얼마나 일치하는지 측정
