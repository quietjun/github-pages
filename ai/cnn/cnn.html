01

CNN이란? — 이미지를 이해하는 신경망

🖼️

이미지는 숫자 행렬

픽셀값(0~255)으로 이루어진 숫자 배열로 인식
컬러 이미지 → R·G·B 3채널

🔍

지역적 패턴 학습

작은 필터(커널)를 슬라이딩하며 적용
엣지·곡선·텍스처 등 지역 특징 자동 추출

📐

위치 불변성

동일 필터를 이미지 전체에 공유 적용
객체가 어디에 있어도 동일 특징으로 인식

💡
MLP와의 결정적 차이
MLP → 이미지를 1D 벡터로 펼쳐 처리 : 공간적 구조(위치 관계) 손실
CNN → 2D 구조 유지 + 인접 픽셀 간 관계를 필터로 학습
파라미터 공유(weight sharing) → 학습 파라미터 수 대폭 절감

nn.Conv2d(in_channels, out_channels, kernel_size, stride=1, padding=0)
01  Convolutional Layer · 합성곱 레이어
입력 이미지에 필터(커널)를 슬라이딩하며 내적(dot product) + bias를 계산해 Feature Map 생성
→ 필터 하나 = 하나의 특징 감지 / 여러 필터를 쌓아 다양한 특징 동시 추출
→ 실제 연산: output = Σ(input × weight) + bias

02  Pooling Layer · 풀링 레이어
Feature Map 크기를 줄여 계산량 절감 + 위치 변화에 강인한 표현 생성
→ 2×2 영역의 최댓값만 선택 : Max Pooling (가장 널리 사용)
→ 학습 파라미터 없음 — 단순 선택/집계 연산 (역전파 시 최댓값 위치만 추적)

nn.Linear(in_features, out_features)
03  Fully Connected Layer · 완전연결 레이어
Conv/Pooling으로 추출된 특징을 1D 벡터로 펼쳐(Flatten) 최종 분류 수행
→ 모든 입력 뉴런이 모든 출력 뉴런에 연결 (완전 연결)
→ 분류 문제의 출력층에는 Softmax로 확률 계산 / 회귀 문제엔 Softmax 불필요

⚠️
출력 활성화 함수 주의
다중 분류: Softmax → 클래스 확률 합 = 1
이진 분류: Sigmoid → 단일 뉴런 출력 (0~1)
회귀: 활성화 함수 없음(None) 또는 linear — Softmax 사용하지 않음
